================================================================================
MATRIX METHOD FORMULAS & NOTATION
================================================================================

STANDARD FORM: Max Z = CX subject to AX = b, X >= 0

NOTATION:
  A     = Coefficient matrix of all variables in constraints (m × n)
  b     = RHS vector (m × 1)
  C     = Objective function coefficient vector (1 × n)
  X     = Decision variable vector (n × 1)

PARTITIONED FORM:
  B     = Basis matrix (coefficients of basic variables) (m × m)
  N     = Non-basis matrix (coefficients of non-basic variables) (m × (n-m))
  Cb    = Objective coefficients of basic variables (1 × m)
  Cn    = Objective coefficients of non-basic variables (1 × (n-m))
  Xb    = Basic variable values (m × 1)
  Xn    = Non-basic variable values (n-m × 1) = 0

KEY FORMULAS:
  1. Xb = B^-1 × b             (Basic variable values)
  2. Zj - Cj = Cb × B^-1 × Aj - Cj  (Reduced costs)
  3. b_new = B^-1 × b          (Updated RHS)
  4. Optimality: All (Zj - Cj) >= 0 for maximization

TABLEAU STRUCTURE:
  +----+-------+----------------+-------+
  | Cb | Basis |  Variables     |  Xb   |
  +----+-------+----------------+-------+
  | Cb | B     | B^-1 × A       |B^-1×b |
  +----+-------+----------------+-------+
  |    | Zj    | Cb×B^-1×Aj     |Cb×Xb  |
  |    | Zj-Cj | Zj - Cj        |  Z    |
  +----+-------+----------------+-------+

WHERE:
  - Each row i shows: B^-1[i,:] × A (updated coefficients)
  - Xb column shows: B^-1 × b (current basic variable values)
  - Zj row shows: Cb^T × (B^-1 × A) for each column
  - Zj-Cj shows optimality indicators
