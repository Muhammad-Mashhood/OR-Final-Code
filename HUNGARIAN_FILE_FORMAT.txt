=================================================================================
HUNGARIAN METHOD INPUT FILE FORMAT
=================================================================================

For Assignment Problems, create a file with the following format:

Line 1: 'min' or 'max'
Following lines: comma-separated cost values (one row per line)

=================================================================================
EXAMPLE 1: 3x3 Minimization Problem
=================================================================================

File content:
min
9,2,7
6,8,3
5,4,8

This represents:
- Minimization problem
- 3 workers x 3 jobs
- Cost matrix:
     J1  J2  J3
  W1  9   2   7
  W2  6   8   3
  W3  5   4   8

Optimal assignment: W1->J2, W2->J3, W3->J1
Total cost: 10

=================================================================================
EXAMPLE 2: 4x5 Maximization Problem (Non-square)
=================================================================================

File content:
max
8,7,6,5,4
9,8,7,6,5
7,6,5,4,3
6,5,4,3,2

This represents:
- Maximization problem
- 4 workers x 5 jobs
- Program will automatically:
  * Add 1 dummy row to balance matrix
  * Convert MAX to MIN problem
  * Find optimal assignment

=================================================================================
EXAMPLE 3: 5x3 Problem (More rows than columns)
=================================================================================

File content:
min
10,15,20
12,18,22
14,16,24
11,17,21
13,19,23

This represents:
- Minimization problem
- 5 workers x 3 jobs
- Program will automatically add 2 dummy columns

=================================================================================
HOW TO USE
=================================================================================

1. Create your cost matrix in problem.txt using the format above
2. Run the program: python final_code.py
3. Select option 8 (Assignment Problem - Hungarian Method)
4. Select option 2 (Load from File)
5. The program will:
   - Read the matrix from problem.txt
   - Balance it if needed
   - Show all steps with visualization
   - Find optimal assignment
   - Calculate total cost

=================================================================================
NOTES
=================================================================================

- Separate values with commas (no spaces needed, but allowed)
- All rows must have the same number of columns
- Values can be integers or decimals
- For MAX problems, program uses Method 1: New Cost = Max Element - Original
- Dummy rows/columns are added automatically for non-square matrices
- Selected zeros are shown with [brackets] in the output
- Lines are marked with -H- (horizontal) or -V- (vertical)
