================================================================================
MATHEMATICAL APPROACH
================================================================================

FOR BASIC VARIABLES:
-------------------
- Variable xi is in basis (non-zero in solution)
- Coefficient Ci appears in both objective and Cb
- When Ci changes, all Zj values change
- Calculate: Zj - Cj = f(C) for each non-basic variable
- Set: f(C) >= 0 for optimality
- Solve: System of inequalities for range

Example:
  Zj-Cj = -1*C + 4 >= 0  →  C <= 4
  Zj-Cj = 1*C - 2 >= 0   →  C >= 2
  Result: 2 <= C <= 4

FOR NON-BASIC VARIABLES:
-----------------------
- Variable xi is not in basis (zero in solution)
- Coefficient Ci only in objective function
- Condition: Zj - Ci >= 0
- Therefore: Ci <= Zj
- Simple upper bound

Example:
  Zj = 5  →  Ci <= 5

EXAMPLE USAGE
================================================================================

PROBLEM:
  max Z = 3x1 + 2x2
  s.t. x1 + x2 <= 4
       2x1 + x2 <= 5

OPTIMAL SOLUTION:
  x1 = 1, x2 = 3, Z = 9

ANALYSIS RESULTS:

Range for C1: 2 <= C1 <= 4
- Current value: 3 (within range)
- Can vary by ±1 without basis change

Range for C2: 1.5 <= C2 <= 3
- Current value: 2 (within range)
- Can vary from 1.5 to 3

Change C1 to 5:
- Outside range [2, 4]
- Becomes non-optimal
- s1 enters basis
- Need new simplex iterations

Change C1 to 3.5:
- Within range [2, 4]
- Stays optimal
- Same solution: x1=1, x2=3
- New Z = 9.5
