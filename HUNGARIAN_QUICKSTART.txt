================================================================================
HUNGARIAN METHOD - QUICK START GUIDE
================================================================================

FEATURE ADDED: Load assignment problems from problem.txt

================================================================================
FILE FORMAT
================================================================================

Line 1: min  (or max)
Line 2+: comma-separated cost values

Example (3x3 minimization):
----------------------------
min
9,2,7
6,8,3
5,4,8

Example (4x5 maximization):
----------------------------
max
8,7,6,5,4
9,8,7,6,5
7,6,5,4,3
6,5,4,3,2

================================================================================
HOW TO USE
================================================================================

1. Create problem.txt with your cost matrix:
   min
   9,2,7
   6,8,3
   5,4,8

2. Run: python final_code.py

3. Select: 8 (Assignment Problem - Hungarian Method)

4. Select: 2 (Load from File)

5. Watch the algorithm solve step-by-step!

================================================================================
FEATURES
================================================================================

[OK] Loads from problem.txt
[OK] Works with any matrix size (3x3, 4x5, 5x3, etc.)
[OK] Automatically balances non-square matrices
[OK] Handles both MIN and MAX problems
[OK] Shows all 7 steps with visualization:
     - Step 1: Balance matrix
     - Step 2: Row reduction
     - Step 3: Column reduction
     - Step 4: Draw minimum lines
     - Step 5: Create additional zeros (if needed)
     - Step 6: Make optimal assignment
     - Step 7: Calculate total cost

[OK] Selected zeros marked with [brackets]
[OK] Lines shown with -H- and -V- markers
[OK] Uses original matrix for final cost calculation

================================================================================
EXAMPLES INCLUDED
================================================================================

problem_hungarian.txt       - 3x3 MIN problem
problem_hungarian_4x5.txt   - 4x5 MAX problem  
test_3x5.txt               - 3x5 MIN problem

test_hungarian_comprehensive.py - Run all tests

================================================================================
NOTES
================================================================================

- Values can be integers or decimals: 5, 10.5, 3.14
- Spaces around commas are optional: "1,2,3" or "1, 2, 3"
- All rows must have same number of columns
- For MAX problems: Uses Method 1 (Max - Original)
- Dummy rows/columns added automatically (cost = 0)

See HUNGARIAN_FILE_FORMAT.txt for detailed format specification.
